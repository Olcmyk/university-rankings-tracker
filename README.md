# 世界大学排名数据集 (World University Rankings Dataset)

**生成日期**: 2026-06-23

本仓库包含三大权威世界大学排名系统的历史数据：

- **QS World University Rankings** (2017-2027)
- **Times Higher Education (THE) World University Rankings** (2011-2026)
- **Academic Ranking of World Universities (ARWU/上海排名)** (2003-2025)
- **Center for World University Rankings (CWUR)** (2012-2015)

## 使用说明

### 数据加载示例

```python
import pandas as pd

# 加载QS 2025年排名
qs_2025 = pd.read_csv('qs-world-rankings-2025.csv')

# 加载THE 2026年排名
the_2026 = pd.read_csv('times2011-2026/outputs/csv/THE_2026_rankings.csv')

# 加载ARWU所有年份数据
arwu_all = pd.read_csv('arwu_all_years_combined_2003-2025.csv')
```

### 主要指标说明

#### QS排名指标

- `Academic Reputation`: 学术声誉
- `Employer Reputation`: 雇主声誉
- `Faculty Student`: 师生比
- `Citations per Faculty`: 教师平均引用率
- `International Faculty`: 国际教师比例
- `International Students`: 国际学生比例
- `International Research Network`: 国际研究网络
- `Employment Outcomes`: 就业成果
- `Sustainability`: 可持续发展

#### THE排名指标

- `teaching`: 教学质量
- `research`: 研究质量
- `citations`: 论文引用
- `international`: 国际视野
- `income`: 产业收入

#### ARWU排名指标

- `alumni`: 校友获奖
- `award`: 教师获奖
- `hici`: 高被引科学家
- `ns`: Nature/Science论文
- `pub`: 国际论文
- `pcp`: 师均表现

## 注意事项

1. 不同排名系统的评价标准和方法不同，排名结果可能存在差异
2. 部分年份的数据可能存在缺失值，使用时请注意处理
3. 排名数据仅供参考，不应作为选择大学的唯一依据
4. 数据来源于公开发布的排名结果，请遵守相应的使用条款
